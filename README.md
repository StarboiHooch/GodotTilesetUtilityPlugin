# Godot Tileset Utility Pluging

This plugin allows you to generate a full tileset of 47 tiles out of a simplified tileset of 9 tiles.
It will also generate a TileSet with Terrain rules set up so you can drag it onto a TileMapLayer and get to work!

## How to import
1. Create a res://addons folder
2. Clone this repo (probably as a submodule) inside that folder
3. Ensure it is enabled in project settings > plugins

## How to use
Once enabled, a new tab will appear in the right side dock. It's quite self-explanatory.
